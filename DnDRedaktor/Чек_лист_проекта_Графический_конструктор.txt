Чек-лист проекта "Графический конструктор"
1. Интерфейс: 
	1.1. Оригинальность цветового решения интерфейса
	1.2. Рабочая область
	1.3. Главное меню (горизонтальный ряд кнопок с названием категорий)
	1.4. Меню графических объектов
	1.5. Функциональное меню (кнопки "Сохранить", "Загрузить", "Снимок экрана", "Справка", "Выход")
2. Функционал:
	2.1. По нажатию на кнопки главного меню появляется меню соответствующих графических объектов
	2.2. По нажатию на кнопки меню графических объектов в рабочей области появляются сами графические объекты
	2.3. Графические объекты можно перемещать:
		2.3.1. Мышкой
	 	2.3.2. Клавишами со стрелками
	2.4. Графические объекты можно увеличивать
		2.4.1. С ограничением размера
	2.5. Графические объекты можно уменьшать
		2.5.1. С ограничением размера
	2.6. По нажатию на кнопку "Сохранить" открывается окно сохранения с возможностью указать место и имя файла
		2.6.1. В указанной папке создается файл .txt
	2.7. По нажатию на кнопку "Загрузить" открывается окно загрузки с возможностью выбрать место и имя файла
		2.7.1. На рабочей области появляется композиция, сохранённая ранее
	2.8. По нажатию на кнопку "Снимок экрана" изображение рабочей области сохраняется в файл .bmp
	2.9. По нажатию на кнопку "Справка" загружается страница с названием и описанием возможностей данной программы
	2.10.  По нажатию на кнопку "Выход" - завершение заботы программы.
	2.11. Возможность добавления графических объектов простым копированием в папку с названием категории
	2.12. Наличие инструкции для добавления графических объектов
3. Дополнения:
	3.1. Наличие начального меню
	3.2. Наличие фоной картинки, соответствующей тематике проекта
	3.3. Смена фона (фоновой картинки)
